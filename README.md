## Projects

### Embedded & Electronics
| Project | Description | Tech |
|---|---|---|
| [Electronic Thermometer](https://github.com/mariokelo/Electronic-thermometer) | Measures temperature using the VBE differential of two NPN transistors. Displays live readings on an OLED screen. | Arduino, Analog Electronics |
| [Wheatstone Bridge Resistance Meter](https://github.com/mariokelo/Wheatstone-Bridge) | Measures unknown resistance by balancing a Wheatstone bridge with a potentiometer. LED and OLED indicate balance state. | Arduino, Analog Electronics |
| [Ultrasonic Radar Scanner](https://github.com/mariokelo/Ultrasonic-radar-scanner-) | Servo-mounted ultrasonic sensor sweeps 150° and streams angle/distance data for real-time radar visualization. | Arduino, Processing |
| [Distance Measurement Sensor](https://github.com/mariokelo/Distance-Measurement) | Ultrasonic distance meter with LCD display and proximity LED alert that blinks faster as objects get closer. | Arduino |
